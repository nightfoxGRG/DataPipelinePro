Конфигуратор

Загрузите файл данных (или несколько файлов из директории) в одном из форматов: .xlsx, .xlsm, .csv. Первая строка должна содержать заголовки колонок. Система переведет названия колонок на английский, определит тип и длину каждой колонки. В результате будет сформирован готовый конфигурационный файл настройки таблиц БД.

Формирование конфига…

Проверьте корректность конфигурационного файла: структуру таблиц, имена колонок, типы и ссылки.

Проверка…

Скачайте шаблонный конфигурационный файл для ручного заполнения или ознакомления со структурой.

Загрузите готовый конфигурационный файл с локального компьютера в систему. Если конфигурационный файл уже существует, он будет заархивирован и заменён новым.

Загрузка файла…

Выгрузите конфигурационный файл текущего проекта из системы на локальный компьютер.
